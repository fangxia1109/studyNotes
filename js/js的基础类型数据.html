• 基础类型
    • 原始类型（基础类型）数据保存在栈内存中
    • 基础类型有：Number、String、Boolean、Undefined、Null、Symbol
      1. Number类型:
        • 数字方法：
          • 整数判断：Number.isInteger()
          • 指定返回的小数位数m：变量.toFixed(m)
        • NaN-表示非数字值
          • isNaN()    //用来判断一个值是否是数字
            //如果该值转换为number等于NaN，则此函数返回true
            注：null; ""两个值会被判断为数字，所以使用前应该屏蔽
        • 浮点精度
          • js语言在浮点数计算时会有精度误差问题
          • 解决方式: toFixed()
      2. String类型:
        • 声明定义
        • 常用字符串方法：
          • 连接运算符：+
          • 获取长度：.length
          • 小写变大写：.toUpperCase()
          • 大写变小写：.toLowerCase()
          • 移除空白：.trim()
          • 获取单字符：.charAt()
          • 截取字符串：.splice(start,end) //start(包含), end(不包含)
          • 查找字符串：.indexOf(需要查找的字符串,第几个)
          • 替换字符串：.repalce(被替换的字符串，替换的字符串)
          • 类型转换：.split(m) //删掉m,分割成数组 + .join(n) //在用n连接数组的每一项
      3. Boolean类型：
        • 声明定义
        • 隐式转换：
          当其他类型与Boolean类型对比时，会将其他类型先转换为数值类型再对比
        • 显式转换： !取反 ！！显示原先的布尔值
      4. Undefined/Null类型：
        • NUll和Undefined基本上是同义的
          null是一个表示"无"的对象，转为数值时为0; undefined是一个表示"无"的原始值，转为数值时为NaN
        • Undefined:
          • 变量被声明了，但没有赋值时，就等于undefined
          • 对象没有赋值的属性，该属性的值为undefined
          • 函数没有返回值时，默认返回undefined
        • Null:
          null表示"没有对象",即该处不应该有值
          用法：
          • 作为函数的参数，表示该函数的参数不是对象
          • 作为对象原型链的终点 null
            Object.getPrototypeOf(Object.prototype)
      5. Symbol(es6的语法)
        • Symbol('name')生成唯一属性
          在一个对象中需要增加一个相同的属性名
  • 复杂类型
    • 复杂类型（引用类型）数据保存在堆内存里通常内存占据空间较大，所以设计将存储内容存放在堆内存，在栈内存里放的是相对应的地址
    • 引用类型：对象{}、数组[]、函数(){}
      • 对象类型：
        • 除了原始值(基本类型),其他(函数、数组)都是对象
        • 语法
          • 在对象里的函数叫方法(methods)
          • 对象里包含属性与值，属性的值可以是任意类型的
        • 对象中的常规操作
          • 增/改
            • 通过"."创建属性
          • 删
            • 通过delete运算符
          • 查
            • 通过"."运算符可以访问对象属性
            • "[]"操作符访问，"[]"里面写字符串
              • 键值对 //键名就是属性名，值就是属性值
        • 对象中的方法怎么访问自己的属性
          • 通过this我们能够访问对象自身属性和方法
        • js中创建对象的几种方式
          • 对象字面量
            • 声明一个对象，赋值给一个变量
          • 构造函数
            • Object自定义构造函数(大驼峰)
              • this指向的是对象的本身
              • 使用new实例化一个对象，就像工厂一样
            • 构造函数的参数应该怎么写
              • 固定参数应用场景
                • 明确知道对象属性
                • 位置要严格对应
              • 对象类型的参数
                • 维护方便
                • 使用方便
