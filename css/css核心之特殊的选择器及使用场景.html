• 特殊选择器

    • 伪类选择器
      • 不改变任何DOM内容。只是插入了一些修饰类的元素
        例：
        • :first-child{} //第一项
        • :last-child{} //最后一项
        • :nth-child(n){} //第n项
        • :nth-child(2n+1){} //奇数项
        • :nth-child(2n){} //偶数项
        • :not{} //偶数项

        • a:link {color:#FF0000} //未访问的链接
        • a:visited {color: #00FF00} //已访问的链接
        • a:hover {color:#FF00FF} //鼠标悬浮后的链接
        • a:active {color:#0000FF} //已选中的链接
    • 伪元素选择器
      • ::first-letter //第一个字母
      • ::first-line //第一行 只能用于块级元素
      • ::selection //选中
      • ::before //在开始位置
      • ::after //在结束位置
