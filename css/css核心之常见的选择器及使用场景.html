选择器

  • 元素（标签）选择器：

    例： div { color: red }
    组合选择器：div, span { color: red }

  • 类选择器：

    • 结合标签选择器:
      例：div.class1 { color:red }
    • 多类选择器:
      例：class="xiaodi background"
    • 链接多个类选择器:
      例：.xiaodi.background{ color:red; background-color:black}

  • id选择器

    • 声明：#import{}
    • 属性：id = "import"
    • 注意：一个id选择器的属性值在html文档中只能出现一次，避免写js获取id时出现混淆

  • 通配符选择器 *

  • 派生选择器

    • 后代选择器
      例：h1 p {color: red;}
    • 子元素选择器
      例：div>span {color: red;}
    • 相邻选择器
      例：h1 + p {color: red;}
